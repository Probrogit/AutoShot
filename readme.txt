Version 1.0 for WoW version 1.12.1

This addon automatically takes a screenshot on certain triggers. Usage: type /as or /autoshot.

Features:

Screenshots on Levelup, Honorable Kill, Battleground Win/Loss, and on a timer.

Known Issues:
Screenshot after BG untested.
